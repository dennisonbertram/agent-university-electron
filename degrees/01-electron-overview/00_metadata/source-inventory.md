# Source Inventory — 01-electron-overview

To be populated during Phase 1 research. Captures every authoritative source consulted, with URL, snapshot date, and reliability rating.

## Categories

- **Official Electron docs** (electronjs.org/docs) — primary
- **Electron source code** (github.com/electron/electron) — secondary, for behavior verification
- **Electron release notes & breaking changes** — version-pinning evidence
- **electron-forge docs + source** — packaging
- **electron-updater docs + source** — auto-update
- **@electron/notarize docs** — notarization
- **better-sqlite3 docs + Electron compat notes** — native module evidence
- **Playwright `_electron` docs** — e2e test evidence
- **Apple developer docs** (notarization, Gatekeeper, entitlements) — signing flow evidence
- **GitHub issues** on `electron/electron` for known gotchas (cited per gotcha in `05_distillation/gotchas/`)
- **ctx7-fetched docs** — for current API surface (preferred over web search)
